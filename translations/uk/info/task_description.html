Панграма (Грецька:παν γράμμα, pan gramma, "кожна літера") або речення, в якому використані всі літери абетки хоча б одного разу кожна. Можливо, вам знайомі деякі з панграм, наприклад, "На подушечці форми любої є й ґудзик щоб пір'я геть жовте сховати." або "Єхидна, ґава, їжак ще й шиплячі плазуни бігцем форсують Янцзи."

Для цього завдання ми будемо використовувати латиницю (A-Z). Дано текст з латинськими літерами та розділовими знаками. Вам потрібно перевірити чи є дане речення панграмою. Регістр літер не має значення.

Вхідні дані: Рядок зі словами (str).

Вихідні дані: Чи є речення панграмою в вигляді логічного значення (bool).

Приклад:

checkPangram("ABCDEF.") == false

check_pangram("The quick brown fox jumps over the lazy dog.") == True
check_pangram("ABCDEF.") == False

Як це використовується: Панграми використовуються для відображення шрифтів, випробування обладнання та для розвитку навичок письма, каліграфії та набору тексту на клавіатурі.
